## Product Evolution

The project started as AI Poster Director, a Web App for generating high-quality prompts from selected styles.

It evolved into the first module of Anson Creative OS, a larger Creative Intelligence Platform designed to learn and amplify a creator's aesthetic.

## Important Lessons

1. Long prompts are intentional.
2. Independent prompts must not inherit unrelated base prompts.
3. Style names must match exactly.
4. Cover file paths must exist.
5. Vercel production updates require Git commit, push, and successful deployment.
6. Avoid manual large paste operations in `page.tsx`; they are error-prone.
